# DSA Practice in Java

Practice repository for Data Structures and Algorithms in Java, with a focus on clear implementations of common interview and problem-solving patterns.

## What's Inside

- `Arrays/` - array traversal, counting, and in-place manipulation problems
- `Stack/` - stack-based problems such as bracket validation

### Current Problems

- `MissingNumber`

- `RemoveDuplicates`
- `RemoveElement`
- `SingleNumber`
- `Balanced_Brackets`
- `Duplicate_Brackets`

## How To Run

Each file is a standalone Java program.

### Compile and run a single file

```bash
javac Arrays/ConsecutiveOnes.java
java -cp Arrays ConsecutiveOnes
```

```bash
javac Stack/Balanced_Brackets.java
java -cp Stack Balanced_Brackets
```

### Compile multiple files

```bash
javac Arrays/*.java Stack/*.java
```

Then run the class you want using the folder as the classpath.
